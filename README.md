# Gigzy 🚀

## Need. Match. Done.

Gigzy is a demand-driven hyperlocal micro-gig marketplace designed for Gen-Z and modern communities. We're trying to bridge the Gap between talent and demand, while helping in reducing unemployment among Indian Youth which is at an all time high(atleast through partial employment/contract via Gigs!

Unlike traditional gig platforms that are supply-focused, Gigzy focuses on real-time user needs by instantly connecting people to nearby help for short-term tasks, errands, collaborations, and digital gigs.

---

## 🌟 Vision

To create a trusted hyperlocal ecosystem where people can:
- Find instant help nearby
- Earn quick money through flexible short gigs
- Build community-driven support networks
- Access both online and offline assistance

---

## 💡 Core Idea

Gigzy bridges the gap between:

### Demand
People needing immediate help:
- Parcel pickup, Pet sitting, Website help, Event support
- Research assistants, Hackathon teammates, Editing videos, Daily errands

### Supply
People with free time, skills, availability, and local accessibility.

---

## 🎯 Target Audience
- Students, Gen-Z users, Hyperlocal communities
- Freelancers, Part-time earners, Busy professionals
- Age Group: 18–40

---

## 🔥 Key Features
- Hyperlocal task matching
- Real-time task posting
- Online + offline gigs
- Verified profiles
- Ratings & reviews
- In-app communication
- Priority task boosting
- Trust & safety systems

---

## 🏗️ Tech Stack

**Backend:** FastAPI, PostgreSQL, Redis, Amazon SQS, AWS Fargate, API Gateway  
**Frontend:** Figma Prototype → React Native (planned)
